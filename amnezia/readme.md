Файл domain.json формируется из v2ray/domains-proxy.txt путем замены начала и концов строк
Многострочную замену можно делать в MS Word
1) Убрать строку с датой
2)  `domain:` -> `    {^p        "hostname":"`
3) `,` -> `",^p        "ip": ""^p    },`
4) Начало файла - `[`
5) Конец файла - `        "ip": ""^p    }"`
6) Конец файла - `]`
7) Заменить угловые кавычки обычными
8) Сохранить файл как *.json
